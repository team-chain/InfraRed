
비상 저장소(Emergency Storage)에서 발견한 물자를 관리하고 생존 시간을 계산하기 위해 제작된 PyQt5 기반의 계산기 프로젝트입니다. 석기 시대에 돌을 깨서 도구를 만들 듯, 현대의 개발 환경에서 Python을 도구 삼아 필요한 시스템을 직접 구축했습니다.

## 1. 프로젝트 개요
이 프로젝트는 아이폰 계산기의 UI 구성과 배치를 최대한 반영하여 설계되었습니다. 긴급한 상황에서도 익숙하게 사용할 수 있는 직관적인 인터페이스를 제공하며, 기본적인 사칙연산을 지원합니다.

## 2. 주요 기능
- **iPhone UI 구현**: 버튼 배치 및 디스플레이 출력 형태를 실제 아이폰 계산기와 유사하게 구성했습니다.
- **사칙연산 지원**: +, -, *, / 연산 및 백분율(%) 계산이 가능합니다.
- **특수 기능**: 
  - `AC`: 계산기 초기화
  - `+/-`: 양수/음수 반전
  - `.`: 소수점 입력
- **예외 처리**: 잘못된 수식 입력 시 'Error' 메시지를 출력하여 시스템 안정성을 확보했습니다.

## 3. 개발 환경 및 제약 사항
- **언어**: Python 3.x
- **라이브러리**: PyQt5 (UI 구현 전용)
- **코딩 스타일**: 
  - [PEP 8](https://peps.python.org/pep-0008/) 가이드를 철저히 준수했습니다.
  - 들여쓰기는 공백 4칸을 사용합니다.
  - 문자열 표현 시 기본적으로 작은따옴표(`' '`)를 사용합니다.
  - 대입 연산자(`=`) 앞뒤로 공백을 두어 가독성을 높였습니다.
